# 学生管理システム (Student Management System)

このプロジェクトは、Java のコンソールアプリケーションであり、学生情報を簡単に管理できます。

## 主な機能
- 学生情報の追加（ID自動割当）
- 学生情報の編集（ID指定）
- 学生情報の削除（ID指定）
- 学生情報の一覧表示
- 名前・GPAでのソート
- データのファイル保存・読み込み

## 技術
- Java (標準ライブラリのみ使用)
- ファイル保存: ObjectOutputStream / ObjectInputStream によるシリアライズ
- データ構造: ArrayList
